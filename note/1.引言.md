# 算法定义

算法定义：给定数据输入，计算满足某种性质输出的问题


```
例子：排序问题

输入：包含 n 个数字的数组
输出：升序排列的数组
```

给定计算问题，算法是一系列良定义的计算步骤

<div style="background-color: #847f7fff;">
良定义：定义明确无歧义

计算步骤：计算机可实现的指令
</div>

插入排序：将数组待排序元素依次插入到已排序的部分，使得已排序部分保持升序性质

选择排序：第一次遍历找到最小的元素，第二次在剩余数组中遍历找到次小元素，....




# 算法性质

- 有穷性：算法必须在有限个计算步骤后终止

- 确定性：算法必须是没有歧义的

- 可行性：可以机械地一步一步执行基本操作

# 算法表示

|---------|---------|---------|

| 伪代码   | 移植编程语言书写形式作为基础和框架，按照接近自然语言的形式表达算法过程，兼顾自然语言和编程语言的优势  |

```python
# 选择排序
from typing import List
def selection_sort(arr: List[int]) -> List[int]:
    """
    选择排序算法
    时间复杂度: O(n²)
    空间复杂度: O(1)
    """
    if not arr:
        return []
        
    n = len(arr)
    for i in range(n - 1):
        # 找到未排序部分的最小值
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        
        # 交换元素
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    
    return arr
```

```python
# 插入排序
def insertion_sort(A: List[int]) -> None:
    """
    插入排序 - 原地排序
    时间复杂度: O(n²)
    空间复杂度: O(1)
    """
    n = len(A)
    for j in range(1, n):
        key = A[j]
        i = j - 1
        
        # 将大于 key 的元素向右移动
        while i >= 0 and A[i] > key:
            A[i + 1] = A[i]
            i -= 1  
        
        # 插入 key 到正确位置
        A[i + 1] = key   
```
# 算法分析

- 归纳基本操作：如运算，赋值，比较

- 统一机器性能：假设基本操作代价都是1
（统一机器性能之后，算法运行时间依赖于问题输入规模与实例）

## 算法分析的原则

- 最好情况：不常出现，不具有普遍性

- 最坏情况：确定上界，更具有一般性

- 一般情况：情况复杂，分析难度大

算法运行时间取决于问题输入规模 $n$ ,表示为 $T(n)$ 


## 算法分析工具

渐进分析：忽略 $T(n)$ 的系数与低阶项，仅关注高阶项，用记号 $\Theta$ 表示

| 渐进记号 | 名称 |
|---------|---------|
| $T(n)=\Theta(g(n))$   | 渐进紧确界   |
| $T(n)=O(g(n))$   | 渐进上界   | 
| $T(n)=\Omega(g(n))$   | 渐进下界  |

### $\Theta$记号

定义：对于给定的函数 $g(n)$，$\Theta(g(n))$ 表示满足以下条件的函数集合：

<!-- $\Theta(g(n))=\{T(n):\exist c_1,c_2,n_0>0,使得\forall n \geq n_0,c_1g(n) \leq T(n) \leq c_2 g(n) \}$ -->
$$
\Theta(g(n)) = \{ T(n) \mid \exists c_1, c_2, n_0 > 0, \text{ 使得 } \forall n \geq n_0, c_1 g(n) \leq T(n) \leq c_2 g(n) \}
$$


### $O$记号

定义：对于给定的函数 $g(n)$, $O(g(n))$ 表示以下的函数集合：


$$
O(g(n)) = \{ T(n) \mid \exists c, n_0 > 0, \text{ 使得 } \forall n \geq n_0, 0 \leq T(n) \leq c \cdot g(n) \}
$$
### $\Omega$记号

定义：对于给定的函数 $g(n)$, $\Omega(g(n))$ 表示以下的函数集合：



$$
\Omega(g(n)) = \{ T(n) \mid \exists c, n_0 > 0, \text{ 使得 } \forall n \geq n_0, 0 \leq c \cdot g(n) \leq T(n) \}
$$

### 渐进分析
 

$$
T(n) = \Omega(g(n)) \quad \text{且} \quad T(n) = O(g(n))
$$
